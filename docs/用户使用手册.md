# EaseSourcer 素材生产工作台 · 使用手册

> 面向实际使用者。本手册讲清楚：这是什么、怎么用、每个操作在哪、有哪些细节要注意。

---

## 一、这是什么

EaseSourcer 是一个**落地页素材生产工作台**：给它一个产品（公司名、品类、描述、产品图），它会自动生成一套 B2B 谷歌广告落地页素材——**banner 大图 + 营销文案 + 多张产品图**，你检查/微调后「标记完成」，即可交付/导出上线。

- 访问地址：`https://easesourcer.omni-marketeer.com`
- 进入需要**访问密码**（向管理员获取）。

---

## 二、主流程总览

```
创建任务  →  自动生成（数分钟）  →  检查 / 微调  →  标记完成  →  交付 / 导出
（两种入口）   （文案+banner+产品图）  （右栏改文案、改图）  （质检闸门）   （拿到两份 JSON / zip）
```

任务在列表页按状态自动分到三个标签：**处理中 → 已完成 → 已归档**。

---

## 三、创建任务（两个入口）

### 入口 1：云工厂后台（主要方式）
在云工厂后台的「Google 投放任务列表」里点 **「创建落地页」** 按钮，任务会自动推送到本平台并开始生成。你**无需在本平台手动建**，直接来列表页看进度即可。

### 入口 2：本平台手动创建
列表页右上角 **「+ 创建落地页」** 按钮 → 弹出表单：

| 字段 | 必填 | 说明 |
|---|---|---|
| 客户公司名 | ✅ | 决定列表里的**显示名** |
| 产品类别 | ✅ | 如「箱包/背包」「机械加工」 |
| 产品描述 | | 卖点、特性、用途（中文即可） |
| 产品中文名 | | |
| 产品英文名 | ✅ | 决定**网址 slug** |
| 上传产品图压缩包(.zip) | | 把产品照片打包上传，越多越好（**多张不同图 → 生成多个不同产品**） |

填好点「创建任务」即开始自动生成。

> 💡 **重要：产品图越多越准。** 只传 1 张图时，系统会把同一产品凑成 3 个「等级变体」，生成的产品图会很相似。要避免，请尽量提供多张不同角度/不同款的图。

## 四、列表页：三个标签与操作

页面顶部三个标签（带数量）：

| 标签 | 含哪些任务 |
|---|---|
| **处理中** | 还没标记完成的（含正在生成、生成失败的） |
| **已完成** | 已点过「标记完成」的 |
| **已归档** | 被归档的（只读） |

每行显示 **「公司名-slug」**，下面一行是 slug、状态、产品数、更新时间。生成中的任务会显示**实时进度条和预计剩余时间**。

每行右侧按钮：
- **打开 / 查看进度 / 查看**：进入工作台详情页（已归档显示「查看」=只读）
- **重试**（仅生成失败时）：重新生成
- **归档**（处理中/已完成）：把任务移到已归档。点击后**按钮旁会弹出确认气泡**，点「确定」生效（防误操作）
- **放回**（已归档）：把任务移回原列表。确认气泡会显示**「将回到：处理中 / 已完成」**

> 说明：本平台**没有删除按钮**——归档就是「移除」的方式，且**随时可放回**，不会丢数据。

## 五、工作台详情页（三栏）

打开任意任务进入工作台，分左/中/右三栏。

### 顶栏
- **← 返回**：回列表
- **code** 与 **✎ slug**：slug 是上线网址标识，点 ✎ 可改（**全局唯一，谨慎改，会影响上线网址**）
- 右上 **✓ 标记完成**（生成后出现）；标记后变为 **已标记完成 · 取消 · 导出 zip**

### 中栏：实时预览
完整落地页预览（banner、文案模块、产品卡片），你的任何改动都会反映在这里。

### 右栏：检查与微调素材
- **文案模块（改 / 删）**：标题、副标题、统计、认证、证言等模块，点「改」编辑、点「−」删除某模块
- **产品文案**：每个产品的名称、描述，可直接改
- **图片产物**：banner、风格参考图、各产品图——**点击任意图片可全屏放大**（支持缩放 +/−、滚轮、拖拽）

> ⚙️ **编辑文案的小细节**：右栏文案输入框是**失焦（点别处）后才同步到预览**，单行输入按**回车**也可。这样打字流畅不卡，不是没生效。

### 左栏：对话改稿 + 输入信息
- **与素材助手对话**：用一句话描述要改什么（如「标题更强调耐腐蚀」「banner 换成车间产线场景」「第 2 个产品图背景太乱，重做」），点「发送」。助手会**先复述理解、列出将执行的待办**，你点「确认执行」它才动手。
- **全部输入内容**（默认收起）：展开可看这个任务的全部原始输入——公司名、昵称、品类、中英文名、描述、排除地区，以及**每张输入原图和它的描述**。右上角有 **✎ 编辑/替换** 按钮（见下一节）。
- **接口 Payload 存档**（默认收起，仅云工厂创建的任务有）：原样展示云工厂传过来的完整数据，排查"信息对不对"时看这里。
- **L2 Prompt 存档 / 生成日志**（默认收起）：技术排查用，一般用户可忽略。

## 六、编辑 / 替换输入（补图片描述、换图）

左栏「全部输入内容」右上角的 **✎ 编辑/替换** 按钮，用于**纠正输入后重新生成**。弹窗里可以：

1. 改公司名、品类、中英文名、产品描述、排除地区
2. **上传新压缩包替换全部图片**：上传后弹窗内会解压并**列出每张图**，你可以**给每张图填一句描述**
3. 点 **「保存并重新生成」** → 用新输入**重新生成全部素材**

> 💡 **图片描述很关键**：它会作为"这张图到底是什么产品"的强信号，纠正只看产品名导致的方向跑偏。如果生成方向不对、或产品图雷同，**补全图片描述 + 多传几张不同图**，再重新生成，效果会明显改善。
>
> ⚠️ 「保存并重新生成」会**覆盖当前的文案和图片**，并**重置「标记完成」**，需要几分钟。取消编辑不会影响原图。

---

## 七、标记完成 → 交付 / 导出

**「标记完成」是质检闸门**：素材检查满意后，点顶栏右上 **✓ 标记完成**。
- 任务从「处理中」流转到「已完成」
- 这之后才能 **导出 zip**，云工厂侧也才能**拉取交付**（两份 JSON：文案 + 产品）
- 如需再改：点「取消」回到可编辑状态

> 没标记完成就导出/交付会被拦截。改完记得重新标记完成。

---

## 八、归档 / 放回

- **归档**：处理中/已完成里点「归档」→ 确认 → 进「已归档」。归档后任务**只读**（打开只能查看，要改先放回）。
- **放回**：已归档里点「放回」→ 确认（会告诉你回到哪个列表）→ 还原。
- 用途：把测试任务、过期任务、已交付不再动的任务收纳起来，保持列表清爽。**完全可逆，不删数据。**

## 九、常见问题 / 注意事项

| 情况 | 怎么办 |
|---|---|
| **多张产品图长得一模一样** | 多半是**只传了 1 张图**。用「✎ 编辑/替换」上传多张不同的图（并填描述）后重新生成 |
| **内容方向跑偏、不像这个产品** | 给图片**补描述**（编辑/替换弹窗里逐张填），描述是比产品名更强的信号 |
| **改了文案预览没变** | 文案是**失焦后**才同步；点一下输入框外面，或单行按回车 |
| **生成失败** | 列表里点「重试」；若反复失败，联系管理员（可能是后端服务波动） |
| **想换上线网址** | 顶栏 ✎ slug 改，但**谨慎**——它决定网址，且全局唯一 |
| **任务不想要了** | 用「归档」收起（可放回）；本平台不提供删除 |
| **想确认云工厂到底传了什么** | 左栏「接口 Payload 存档」看原始数据 |

---

## 十、一句话速记

> **来列表页 → 打开任务 → 中栏看预览、右栏改文案改图 / 左栏对话改稿 → 满意了点「标记完成」→ 交付。**
> 生成不理想？**多传图 + 补图片描述 → 编辑/替换 → 重新生成。**
